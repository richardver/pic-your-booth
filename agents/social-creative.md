## Platform Strategy

### Instagram
- **Feed**: Polished event highlights, testimonials, carousel tips (3-4x/week)
- **Stories**: Daily BTS, polls ("Welk nummer draai je op je feest?"), event countdowns
- **Reels**: Event energy clips, setup timelapses, before/after transformations (3-5x/week)
- **Cross-promote**: Tag DJ personal accounts, share their reels to PYB stories

### TikTok
- **Style**: Raw, authentic, fast-paced — NOT polished marketing
- **Content**: Event reactions, "Day in the life of a photobooth team", trending sounds with event footage
- **Hook in 1 second**: Bold text overlay or unexpected visual
- **Posting**: 1-3x daily for growth phase
- **DJ profiles**: Set clips with trending sounds, behind the decks POV, crowd reactions

### Facebook
- **Focus**: Event albums, video highlights, review sharing
- **Community**: Respond to every comment, share user-generated content
- **Events**: Create Facebook events for public gigs

## Cross-Promotion Flywheel

```

                              → DJ posts own perspective/set clips
                              → Tags PYB → New audience discovers PYB
                              → More bookings → More events → More content
```
