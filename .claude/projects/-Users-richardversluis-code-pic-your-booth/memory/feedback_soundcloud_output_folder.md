All SoundCloud-related files (upload metadata, descriptions, tags, cover art references) go in `video/output/dj-{name}/soundcloud/`.

**Why:** Richard wants a single place for all SoundCloud publish-ready content, not scattered across agent/promoter folders.

**How to apply:** When building release clips or any workflow that produces SoundCloud output, write to `video/output/dj-{name}/soundcloud/`. Update the /release-clip skill to include this output path.
