# Video Ad Script Writer

Write scroll-stopping scripts for short-form video ads that convert.

## When to Use

- Creating TikTok, Reels, or Shorts ads
- Writing YouTube pre-roll scripts
- Briefing UGC creators
- Planning video creative strategy
- Refreshing video content

## Input Requirements

Provide:
1. **Product/Service**: What you're advertising
2. **Target Audience**: Who you're speaking to
3. **Platform**: TikTok, Reels, YouTube, Meta
4. **Length**: 15s, 30s, 60s
5. **Style**: UGC, professional, talking head, demo
6. **Goal**: Awareness, consideration, conversion
7. **Key Message**: One thing viewers should remember

## Video Length Guidelines

| Platform | Recommended | Max |
|----------|-------------|-----|
| TikTok | 15-30s | 60s |
| Instagram Reels | 15-30s | 90s |
| YouTube Shorts | 15-30s | 60s |
| Meta Feed | 15-30s | 240s |
| YouTube Pre-roll | 6s or 15-30s | 15-30s |

**Rule of thumb**: Shorter is almost always better

## Script Structure Framework

### The 3-Part Video Ad Formula

```
HOOK (0-3 seconds)
↓
BODY (3-20 seconds)
↓
CTA (last 3-5 seconds)
```

### Part 1: The Hook (0-3 Seconds)

**Goal**: Stop the scroll, create curiosity

**Hook Types**:

| Type | Example | Best For |
|------|---------|----------|
| Bold Claim | "This replaced my $200 product" | Disruptive products |
| Question | "Why does everyone's skin look like this?" | Problem-aware audience |
| Call Out | "POV: You're still using [old thing]" | Direct response |
| Pattern Interrupt | Start with unexpected visual/sound | Awareness |
| Controversy | "Unpopular opinion: [take]" | Engagement |
| Before/After | Show transformation immediately | Results-based |
| Story Opening | "So there I was at 3am..." | UGC style |

### Part 2: The Body (3-20 Seconds)

**Goal**: Deliver value, build desire

**Body Structures**:

**Problem-Solution**:
```
[Show/describe problem]
↓
[Introduce your solution]
↓
[Show it working]
```

**Demonstration**:
```
[Show product in action]
↓
[Highlight key feature]
↓
[Show result]
```

**Testimonial/Story**:
```
[Relatable situation]
↓
[Discovery of product]
↓
[Transformation/outcome]
```

**Educational**:
```
[Share valuable insight]
↓
[Connect to product]
↓
[Show benefit]
```

### Part 3: The CTA (Last 3-5 Seconds)

**Goal**: Tell them exactly what to do

**CTA Types**:
- Direct: "Click the link to get yours"
- Soft: "Link in bio if you want to try it"
- Urgency: "Sale ends tonight - link below"
- Social proof: "Join 10,000 others - link in bio"

## Script Templates by Style

### Template 1: UGC Talking Head (30s)

```
[HOOK - 0-3s]
(Looking at camera, mid-sentence energy)
"Okay I have to talk about this because [relatable problem]..."

[BODY - 3-25s]
"I've tried like [number] different [solutions] and nothing worked until I found [product]."

(Show product)
"What makes this different is [key feature]."

(Demo or result)
"And look - [show proof/result]."

[CTA - 25-30s]
"Seriously if you struggle with [problem], try this. Link in my bio."
```

### Template 2: Problem-Agitation-Solution (15s)

```
[HOOK - 0-3s]
(Text on screen + visual)
"Stop doing THIS with your [thing]"

[BODY - 3-12s]
"It's actually causing [problem]."

(Quick transition)
"Instead, use [product/solution]."

(Show it working)
"See the difference?"

[CTA - 12-15s]
"Link in bio to get yours."
```

### Template 3: Before/After Demo (30s)

```
[HOOK - 0-3s]
(Split screen or quick cut)
Before: [problem state]
After: [result state]

[BODY - 3-25s]
"Here's exactly what I did..."

(Step-by-step demo)
"Step 1: [action]"
"Step 2: [action]"
"Step 3: [action]"

(Show final result)
"And that's it."

[CTA - 25-30s]
"Want to try it? Link below."
```

### Template 4: Listicle/Tips (30s)

```
[HOOK - 0-3s]
"3 things I wish I knew about [topic]"

[BODY - 3-25s]
"Number 1: [tip + visual]"
(3-5 seconds)

"Number 2: [tip + visual]"
(3-5 seconds)

"Number 3: [tip + product integration]"
(5-7 seconds)

[CTA - 25-30s]
"For more tips, follow. Link in bio for [offer]."
```

### Template 5: Story Arc (60s)

```
[HOOK - 0-3s]
"I never thought I'd be the person to [outcome]..."

[SETUP - 3-15s]
"Six months ago, I was [problem state]."
"I had tried [other solutions] and nothing worked."
"I was about to give up when..."

[TURNING POINT - 15-35s]
"...I found [product/solution]."
"At first I was skeptical because [objection]."
"But then I [tried it/saw results]."

(Show proof/transformation)

[RESOLUTION - 35-50s]
"Now, [new state/outcome]."
"The thing that made the difference was [key benefit]."

[CTA - 50-60s]
"If you're where I was, try this. Link in bio - you won't regret it."
```

## Platform-Specific Adjustments

### TikTok
- Native, raw aesthetic
- Trending sounds (when relevant)
- Text overlays throughout
- Fast cuts/transitions
- Reply to comment format works well

### Instagram Reels
- Slightly more polished than TikTok
- Can be more aesthetic
- Music-forward often works
- Strong visual hooks

### YouTube Shorts
- Can be more informational
- Slightly longer attention span
- Educational content performs well

### Meta Feed Video
- Sound-off optimization critical
- Captions required
- More direct selling accepted
- Can be longer (30-60s)

### YouTube Pre-roll
- Hook must work in first 5 seconds (skip button)
- Front-load the value
- Clear branding early
- Consider 6-second bumper ads

## Hook Swipe File

### For Physical Products
- "I'm convinced this is the best $[X] I've ever spent"
- "Why did nobody tell me about this sooner"
- "Unpopular opinion: [cheap thing] works better than [expensive thing]"
- "Watch me use this for the first time"
- "This thing has been sold out for weeks - here's why"

### For Services/Apps
- "This app is literally free and does [valuable thing]"
- "I saved [X hours/dollars] last month with this"
- "Stop paying for [thing] - here's what I use instead"
- "The tool that changed how I [activity]"

### For Courses/Info Products
- "The [thing] they don't teach you in [school/industry]"
- "I learned this the hard way so you don't have to"
- "Why [common advice] is actually wrong"
- "The real reason [problem] keeps happening"

### For B2B/Software
- "If you're still using [old way], watch this"
- "This is how [impressive company] does [thing]"
- "I automated [task] and here's what happened"

## Questions to Ask

1. What's the single most compelling benefit?
2. What transformation can you show?
3. Do you have before/after proof?
4. What objections should we address?
5. Who will be on camera (talent/founder/UGC)?
6. What's worked before? What hasn't?
