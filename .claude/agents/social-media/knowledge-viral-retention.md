# Viral Loops & Retention

Cross-promotion, referrals, and post-event engagement. Full reference in `docs/marketing/marketing-psychology-viral-retention.md`.

---

## The PYB Viral Loop

```
Guest experiences photobooth →
Photos shared on social (tagged/branded) →
Friends see and ask "where was this?" →
Guest mentions PYB →
New customer discovers PYB →
Books for own event →
More guests experience photobooth → ...
```

## Viral Assets

| Asset | Mechanism |
|---|---|
| Printed photos | Physical keepsake with PYB branding |
| Digital photo strips | Shared via WhatsApp/Stories with watermark + link |
| Event highlight reels | Posted on PYB socials, shared by hosts |
| AI photobooth results | Novelty drives sharing |
| DJ set clips | Posted on DJ socials, tagged to PYB |

## Post-Event Content Sequence

| Timing | Action | Goal |
|---|---|---|
| Day 0 | Capture content at event | Social content |
| Day 1 | Thank-you post/story tagging host | Engagement |
| Day 3 | Photo gallery link shared | Re-trigger positive emotions |
| Day 5 | Ask for Google review | Social proof |
| Day 7 | Referral offer post | Activate viral loop |

## Referral Program

- Referrer: €50 korting op volgende boeking
- Referee: €50 korting op eerste boeking
- Best trigger moments: post-event, gallery delivery, review request

## Hook Model for Followers

| Step | Implementation |
|---|---|
| Trigger | Reel in discover, TikTok algorithm, retargeting |
| Action | Watch video (low effort) |
| Variable Reward | Different events, energy, reactions each time |
| Investment | Follow, save, comment → algorithm shows more |
