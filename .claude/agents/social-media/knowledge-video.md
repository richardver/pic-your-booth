## Hook Techniques (First 1-3 Seconds)

1. **Text hook on screen** — bold statement before speaking
2. **Direct address** — "Here's something nobody tells you about..."
3. **Pattern interrupt** — unexpected visual or action
4. **Question** — "Did you know...?"
5. **Trending sound** — leverage current audio trends

## Script Structures

**AIDA:** Attention → Interest → Desire → Action
**PAS:** Problem → Agitate → Solution
**Before-After-Bridge:** Current state → Desired state → How to get there

## Production Tier for PYB

Tier 1 — Phone + Remotion (best for TikTok, Reels, Stories):
- Smartphone camera (iPhone 13+)
- Lavalier or wireless mic
- Natural light or ring light
- Edit & overlay in Remotion (`npx create-video@latest --tiktok`)
- Brand-consistent templates as React components with DJ design tokens

## PYB Clip Selection Criteria

| Criteria | Why |
|---|---|
| Guest reactions ("wow" moments) | Most shareable |
| Photo result reveals | Shows product quality |
| Dance floor energy | Sells DJ combo |
| Setup timelapse | Satisfying, shows professionalism |
| Customer testimonial | Social proof, converts |
| Behind the decks (DJ POV) | Authentic, TikTok-native |

## Video Length by Platform

| Platform | Optimal | Max |
|---|---|---|
| TikTok | 15-60s | 10 min |
| Instagram Reels | 15-30s | 90s |
| Instagram Stories | 15s/slide | 60s |
| Facebook Feed | 30-120s | 240 min |

## Content Repurposing Waterfall

```
Event Highlight Video (2-3 min)
├─→ 3-5 Short Clips → TikTok, Reels, Shorts
├─→ Key Frames → Quote graphics for Instagram
├─→ BTS clips → Stories, TikTok
├─→ DJ set clips → DJ personal accounts
├─→ Best photos → Instagram carousel
└─→ Testimonial clips → Website, ads, social proof
```
