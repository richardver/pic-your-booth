# On-Page SEO

Heading structuur, keyword plaatsing, interne links en content regels.

---

## Heading Hierarchie

Elke pagina moet deze structuur volgen:

```
H1: Primair keyword (1x per pagina, nooit meer)
  H2: Sectie koppen (bevat long-tail keywords waar natuurlijk)
    H3: Sub-secties
```

|---|---|---|

## Keyword Plaatsing

| Locatie | Prioriteit | Regel |
|---|---|---|
| H1 | KRITISCH | Bevat primair keyword |
| Meta title | KRITISCH | Begint met primair keyword |
| Meta description | HOOG | Bevat keyword + CTA |
| Eerste alinea | HOOG | Keyword in eerste 100 woorden |
| H2 koppen | MIDDEL | Long-tail keywords, niet forceren |
| Alt text afbeeldingen | MIDDEL | Beschrijvend + keyword waar relevant |
| URL slug | HOOG | Keyword in URL |
| Interne links anchor text | MIDDEL | Beschrijvend, niet "klik hier" |

## Keyword Dichtheid

- Primair keyword: 1-2% (niet meer)
- Synoniemen en variaties gebruiken: "photobooth" / "fotobooth" / "fotobox" / "fotospiegel"
- Natuurlijk Nederlands. Nooit keyword-stuffing.

## Interne Link Structuur

```
Homepage
├── /magic-mirror (vanuit service card + nav)
├── /party-booth (vanuit service card + nav)
├── /djs (vanuit service card + nav)
└── /offerte (vanuit nav CTA + alle product CTAs)

Product Pagina's
├── /offerte (vanuit package builder CTA)
└── Homepage (vanuit nav logo)
```

Elke productpagina linkt naar /offerte via de CTA. Homepage linkt naar alle productpagina's. Circulaire structuur met offerte als eindpunt.

## Image Alt Text Regels

| Type | Format | Voorbeeld |
|---|---|---|
| Product foto | [product] [context] PicYourBooth | "Magic Mirror XL fotospiegel op bruiloft PicYourBooth" |
| Event foto | [activiteit] [locatie/type] | "Gasten bij photobooth op bedrijfsfeest" |

| Decoratieve afbeelding | alt="" (leeg) | Achtergrondafbeeldingen |

## Content Lengte Richtlijnen

| Pagina type | Minimale woordentelling | Focus |
|---|---|---|
| Productpagina | 500+ woorden | USPs, FAQ, package builder content telt mee |
| Homepage | 300+ woorden | Service overzicht, social proof |
| Offerte | Minimaal | Formulier is de content |
| Stadspagina (Fase 2) | 800+ woorden | Lokale relevantie, testimonials uit die stad |
