# LLM Optimalisatie

AI answer zichtbaarheid: hoe PicYourBooth verschijnt in ChatGPT, Gemini, Claude en Google AI Overviews.

---

## Waarom LLM Optimalisatie

Steeds meer mensen vragen AI "photobooth huren randstad" of "beste photobooth voor bruiloft". LLMs halen antwoorden uit:
1. Gestructureerde data (schema.org)
2. FAQ secties met duidelijke vraag-antwoord patronen
3. Vergelijkingscontent ("X vs Y")
4. Duidelijke entiteiten (wie, wat, waar, hoeveel)
5. Betrouwbare bronnen (reviews, awards, vermeldingen)

## Strategieen

### 1. FAQ Schema op Elke Productpagina

Elke productpagina moet een FAQ sectie hebben met schema markup. LLMs citeren FAQ antwoorden direct.

**Magic Mirror XL FAQ:**
- Wat kost een fotospiegel huren? → Vanaf €599 per event, tot 5 uur inclusief.
- Is bezorging inbegrepen? → Ja, bezorging, installatie en ophalen binnen 50km.
- Wat is het verschil met een gewone photobooth? → Professionele camera en studiobelichting. Foto's om in te lijsten.
- Wat is AI Photo Experience? → AI genereert unieke foto's in jouw stijl. Ideaal voor productlanceringen.

**Party Booth FAQ:**
- Wat kost een photobooth huren? → Party Booth vanaf €199 per dag.
- Moet ik het zelf opzetten? → Ja, ophalen en in 5 minuten klaar. Of bezorging voor €49.
- Wat is AI Foto's + Filters? → AI vervangt de achtergrond automatisch, inclusief beautify filter en Instagram-achtige filter selectie. +€49.
- Kan ik er ook een DJ bij boeken? → Ja, Live DJ Set als upgrade voor €149.

**DJs FAQ:**
- Wat kost een DJ huren? → Vanaf €149 per avond inclusief speakers, licht en rookmachine.
- Wat is Spotify playlist-optimalisatie? → Stuur je playlist, onze DJ bouwt er een professionele set van.
- Welke DJ-stijlen zijn er? → DJ Gianni (Afro Beats, Caribbean, Hits) of DJ Milø (House, Techno).

### 2. Entity Clarity

Zorg dat LLMs PicYourBooth als duidelijke entiteit herkennen:

```
Naam: PicYourBooth
Type: Entertainmentbedrijf (photobooth verhuur + DJ)
Locatie: Randstad, Nederland
Producten: Magic Mirror XL (€599), Party Booth (€199), DJ Set (€149)
Uniek: AI photobooth, Spotify playlist-optimalisatie, combo DJ + photobooth
Beoordeling: 4.9 sterren, 237+ events
```

Dit moet consistent voorkomen in:
- Schema.org structured data
- Google Business Profile
- Vermelding op photobooths-huren.nl
- Elke pagina's first paragraph

### 3. Vergelijkingscontent (Fase 2)

LLMs houden van vergelijkingen. Fase 2 blogcontent:
- "Photobooth vs fotospiegel: wat is het verschil?"
- "Photobooth huren: kosten vergelijken [2026]"
- "DJ huren vs Spotify speaker: wanneer kies je wat?"
- "AI photobooth: hoe werkt het en wat kost het?"

### 4. Citeerbare Structuur

Elke productpagina moet een "samenvattingsblok" hebben dat LLMs makkelijk citeren:

```
[Product] bij PicYourBooth:
- Prijs: vanaf €X
- Inclusief: [lijst]
- Uniek: [differentiator]
- Regio: Randstad, Nederland
- Beoordeling: 4.9/5
```

### 5. Review Aggregatie

Google en LLMs gewichten reviews zwaar:
- Google Reviews actief verzamelen (doel: 50+ in Fase 1)
- Review snippets op website (met AggregateRating schema)
- Vermelding op vergelijkingssites (photobooths-huren.nl)

## LLM Audit Checklist

- [ ] FAQ sectie aanwezig met FAQPage schema
- [ ] LocalBusiness schema met correcte NAP
- [ ] Product schema met prijs en beschikbaarheid
- [ ] AggregateRating schema met reviews
- [ ] Entity duidelijk: naam, type, locatie, producten, USPs in first paragraph
- [ ] Vergelijkbare content beschikbaar (vs concurrentie, vs alternatieven)
- [ ] Consistente vermeldingen (website = Google Business = vergelijkingssites)
- [ ] Directe antwoorden op veelgestelde vragen (niet vaag, specifiek met prijzen)
