Promotie Playbook

TIKTOK PROMOTIE

Van SoundCloud mixtape naar TikTok views naar boekingen. Clip-voor-clip scripts, storyboards en caption templates. Herhaalbaar systeem voor elke release.

Het Systeem

ELKE MIXTAPE = 7 CLIPS

Elke SoundCloud mixtape release krijgt een promotiecyclus van 7 dagen op TikTok. 7 clips die elk een ander aspect van de set belichten. Film alles tijdens het opnemen van de set - dat is de meest authentieke content.
WAAROM 7 DAGEN?

TikTok's algoritme beloont consistente posting. 7 clips in 7 dagen geeft het algoritme genoeg data om je te pushen. Na dag 3-4 weet TikTok wie je content wil zien.

FILM TIJDENS OPNAME

Zet je telefoon op statief achter of naast je controller. Film de hele sessie. Knip later de beste momenten eruit. 2 uur opname = 7+ clips materiaal.

ALTIJD LINK NAAR SET

Elke clip is een advertentie voor de volledige set op SoundCloud. "Link in bio" of "Volledige set op SoundCloud" in elke caption. Nooit een clip zonder doorverwijzing.

EDIT MET REMOTION

React-based video templates. Definieer elke clip als een herbruikbare component met DJ Gianni's exacte design tokens (coral, Bebas Neue, EQ-bar). Zelfde data, nieuw genre = nieuwe clip. Exporteer op 1080x1920 (9:16). npx create-video@latest --tiktok

Gouden Regel: Hook in 1 Seconde

TikTok gebruikers scrollen in 0,3 seconden langs je video. Je eerste frame moet actie zijn: handen op de controller, een beat drop, tekst op het scherm. Nooit beginnen met "Hey guys" of een stil beeld. De muziek IS de hook.

Dag 0

RELEASE CLIP

De beste drop uit de set - dit is de hoofdclip
15-30 sec 9:16 verticaal Streams driven Behind the decks
Remotion template — automatisch gerenderd door video-editor agent
0:00
Hook: POV tekst slamt in (SlamIn 1.3x). *Accent* woord in coral. Genre pills wipen in na 1s. Camera: SlowZoomDrift 1.0→1.04. Geen statisch beeld.
0:03
Build-up van de track. CoralVignette bouwt spanning op. SlowZoomDrift 1.0→1.06. Clean footage — geen overlays, muziek domineert.
0:06
Pre-drop: Shake groeit van 0→4px, brightness dipt, vignette intensifieert. Visuele spanning bouwt mee met de audio. Iets gaat gebeuren.
0:08
DE DROP. BassFlash 50% + ParticleBurst + dubbele flash. DJ GIANNI + serie naam slamt in als een blok (180px). 4 seconden op scherm, fade out. Alles vuurit tegelijk. PostDropDecay: aftershock shake + scale settle.
0:10
Vibe sectie: SlowPan (drift rechts), warmere kleurgrade. Muziek domineert. Booking CTA (vanaf 12s): "BOEK DJ GIANNI" + "Populaire data gaan snel" + coral button. DeckHits bij 16-20s: 5 zoom punches (1.12x) + flash per hit bij deck actie.
0:20
End card: Cover art 680px met Ken Burns zoom + serie titel + booking CTA. Audio loopt door tot het einde. Geen stilte.
Rendering: De video-editor agent rendert dit automatisch via Remotion met audio-analyse voor drop-timing. Template: video/templates/tiktok/release-clip/

TikTok vs Instagram Hashtag Regels

TikTok: 5-10 hashtags. Minder is meer. TikTok leest je video content (audio, tekst, visuals) om te categoriseren - hashtags zijn een hulpmiddel, niet de enige manier.

Instagram: 20-30 hashtags. Meer is meer. Instagram leunt zwaarder op hashtags voor discovery. Voeg bredere tags toe (#feest, #party, #nederland, #dj) naast de niche tags.

Video Tooling

REMOTION WORKFLOW

We gebruiken Remotion in plaats van CapCut. Remotion is een React-framework voor programmatische video's. Dat betekent: maak een template een keer, gebruik het voor elke release met andere data. Consistent, schaalbaar, on-brand.
WAAROM REMOTION?

Herbruikbare templates: Maak een "Release Clip" component een keer met DJ Gianni's design tokens (coral, Bebas Neue, SlowZoomDrift, DeckHits). Volgende keer: zelfde template, ander genre en andere tracks. Geen handmatig knippen meer.

BRAND CONSISTENTIE

Elke clip gebruikt exact dezelfde kleuren, fonts en animaties als de cover art en website. Geen "bijna goed" meer - het is pixel-perfect DJ Gianni, elke keer.

TIKTOK TEMPLATE

Remotion heeft een dedicated TikTok starter: npx create-video@latest --tiktok. 9:16 formaat, word-by-word captions via Whisper.cpp, en Tailwind CSS support uit de doos.

RENDER & EXPORT

Render via CLI (npx remotion render), in de browser (Remotion Studio), of server-side. Exporteert direct naar MP4. Geen watermarks, volledige controle over kwaliteit.

PROJECT SETUP

Remotion TikTok Project Opzetten
1
Maak een nieuw Remotion project: npx create-video@latest --tiktok Dit installeert Remotion met TikTok template, Whisper.cpp voor captions, en Tailwind CSS.
2
Open Remotion Studio: npx remotion studio Preview je video in de browser. Live reload bij code changes.
3
Pas de Composition aan voor TikTok: width={1080} height={1920} fps={30} 9:16 verticaal formaat, 30fps is standaard voor TikTok.
4
Render naar MP4: npx remotion render src/index.ts clip-name out/clip.mp4 Of gebruik de render-knop in Remotion Studio.

CLIP TEMPLATES ALS REACT COMPONENTS

Elk clip type wordt een herbruikbare React component. Verander de props (genre, tracklist, kleuren) en je hebt een nieuwe clip. Hier zijn de 7 templates die je nodig hebt:
Component Props Animaties Duur
<TeaserClip /> genre, serieNaam, coverArt, audioSnippet Fade-in tekst, cover art zoom, EQ-bar pulse 10-15s (300-450 frames)
<ReleaseClip /> genre, serieNaam, videoFile, dropTimestamp Text overlay, speed ramp op drop, fade-to-cover 15-30s (450-900 frames)
<TransitionClip /> genre, videoFile, hookText Text hook animatie, pull-back reveal 15-20s (450-600 frames)
<GuessTrackClip /> genre, videoFile, cutTimestamp Audio cut, freeze frame, vraagteken animatie 10-15s (300-450 frames)
<TracklistClip /> genre, serieNaam, tracks[], accentColor Scroll-up tracklist, coral nummering, staggered appear 20-30s (600-900 frames)
<BTSClip /> genre, videoFile, steps[] Step-by-step text overlays, audio switches 30-60s (900-1800 frames)
<FavoriteClip /> genre, videoFile, persoonlijkeTekst Tekst overlay fade, vibing footage loop 15-25s (450-750 frames)
Design Tokens in Remotion

Gebruik DJ Gianni's design tokens als CSS variabelen of een shared config in je Remotion project:

--gi: #f0654a (coral/Afro Beats), --caribbean: #f5b731 (golden amber), --urban: #c084fc (warm purple)
font-family: 'Bebas Neue' (titels), 'Space Grotesk' (body)

De accentkleur wordt bepaald door de genre prop. Eenzelfde component werkt voor alle drie de series.

Behind-the-Decks Footage + Remotion Overlay

De workflow is hybride: film de behind-the-decks footage met je telefoon (raw video), importeer in Remotion als <OffthreadVideo />, en voeg de branded text overlays, EQ-bars en cover art er programmatisch overheen. Zo combineer je authentieke footage met pixel-perfect branding.

Productie

FILM SETUP

Je hebt niet meer nodig dan een telefoon en een statief. Hier is precies hoe je filmt tijdens het opnemen van je set zodat je genoeg materiaal hebt voor alle 7 clips.
CAMERA SETUP

Positie 1 (hoofd): Telefoon op statief, achter je of schuin naast je. Gericht op je handen + controller. Dit is je "behind the decks" angle - 80% van je clips komt hieruit.

TWEEDE HOEK (OPTIONEEL)

Positie 2: Tweede telefoon of GoPro op de tafel, low angle omhoog gericht. Geeft variatie in je edits. Niet verplicht voor de eerste mixes, maar maakt de content sterker.

BELICHTING

Donkere kamer + 1 lichtbron (bureaulamp, ringlight, of LED strip achter je scherm). Het moet voelen als een nachtclub, niet als een kantoor. De controller mag oplichten, jij zit in de schaduw.

AUDIO

Film met telefoon audio (de controller/speakers geven genoeg geluid). In de edit: vervang audio met de WAV/FLAC van de set voor optimale kwaliteit. Sync op de drop.

Minimale Setup Checklist

☐ Telefoon (iPhone 13+ of vergelijkbaar)

☐ Donkere kamer
☐ 1 lichtbron (warm of coral-tint)
