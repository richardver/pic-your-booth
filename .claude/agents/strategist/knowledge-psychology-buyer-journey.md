# Marketing Psychology - Buyer Journey & Decision Making

## Jobs-to-Be-Done Framework

People don't buy a photobooth - they hire it to do a job.

### PYB Job Statements

**Magic Mirror XL:**
```
When I'm planning a special event (wedding/corporate/high-end party),
I want to create an unforgettable experience for my guests,
so they talk about my event for weeks and I look like an amazing host.
```

**Party Booth:**
```
When I'm organizing a celebration on a budget,
I want easy, fun photo entertainment without the hassle,
so my guests have a great time and I have memories to keep.
```

**DJ + Photobooth Combo:**
```
When I'm planning a party and need music AND entertainment,
I want to book everything from one team,
so I have less stress, one point of contact, and a guaranteed great vibe.
```

### Four Forces of Progress (Switching)

For someone to book PYB instead of doing nothing or going with a competitor:

| Force | Direction | PYB Application |
|---|---|---|
| **Push** (pain with current) | Toward switching | "Last event had no entertainment and guests left early" |
| **Pull** (appeal of new) | Toward switching | "Wow-factor photobooth + DJ that reads the crowd" |
| **Anxiety** (fear of new) | Against switching | "What if the equipment breaks?" "What if the DJ is bad?" |
| **Habit** (comfort with old) | Against switching | "We've never had a photobooth before" / "We always use DJ X" |

**Switching happens when**: Push + Pull > Anxiety + Habit

**Reduce anxiety with**: Testimonials, guarantee, video of real events, "vrijblijvend voorstel"
**Break habit with**: Comparison content, unique selling points, first-time offer

## Customer Journey Map - PYB

### Stage 1: Awareness
| Element | Detail |
|---|---|
| **Customer goal** | "I need entertainment for my event" |
| **Touchpoints** | Meta ad, Instagram reel, Google search, word-of-mouth |
| **Emotion** | Curious (+2), maybe overwhelmed by options |
| **Content need** | Eye-catching event visuals, quick value prop |
| **PYB action** | Meta ads with event footage, Instagram content, SEO |

### Stage 2: Consideration
| Element | Detail |
|---|---|
| **Customer goal** | "What are my options? What fits my event?" |
| **Touchpoints** | Website visit, package comparison, reviews |
| **Emotion** | Interested (+3), evaluating |
| **Content need** | Clear packages, pricing, testimonials, event gallery |
| **PYB action** | Landing page with social proof, package comparison |

### Stage 3: Decision
| Element | Detail |
|---|---|
| **Customer goal** | "Is this the right choice? Can I trust them?" |
| **Touchpoints** | Proposal form, proposal email, follow-up |
| **Emotion** | Hopeful (+3) but anxious (-1) about commitment |
| **Content need** | Detailed proposal, reviews, guarantee, FAQ |
| **PYB action** | Fast proposal response (<24h), personal touch, address objections |

### Stage 4: Booking
| Element | Detail |
|---|---|
| **Customer goal** | "I want to confirm and stop worrying" |
| **Touchpoints** | Booking confirmation, payment |
| **Emotion** | Excited (+4), relieved |
| **Content need** | Clear next steps, what to expect |
| **PYB action** | Smooth booking flow, confirmation email with event prep info |

### Stage 5: Pre-Event
| Element | Detail |
|---|---|
| **Customer goal** | "I want to know everything is handled" |
| **Touchpoints** | Pre-event email, Spotify playlist discussion (if DJ booked) |
| **Emotion** | Anticipating (+3), slightly nervous |
| **Content need** | Timeline, what PYB will bring, contact for questions |
| **PYB action** | Pre-event check-in email, confirm logistics, playlist optimization |

### Stage 6: Event Day
| Element | Detail |
|---|---|
| **Customer goal** | "My guests should have the time of their lives" |
| **Touchpoints** | On-site setup, photobooth experience, DJ performance |
| **Emotion** | Peak moment (+5) - "Wow, that was an experience" |
| **Content need** | None - they're living it |
| **PYB action** | Professional setup, high-energy delivery, capture content for social |

### Stage 7: Post-Event (Advocacy)
| Element | Detail |
|---|---|
| **Customer goal** | "Share the memories, maybe book again" |
| **Touchpoints** | Photo gallery delivery, review request, social media |
| **Emotion** | Happy (+4), nostalgic |
| **Content need** | Easy access to photos, shareable moments |
| **PYB action** | Send photos, ask for Google review, share event on socials (with permission) |

**Key insight**: The dip at Decision (-1, anxiety) is where most leads drop off. This is where you need the strongest trust signals, social proof, and risk-reversal.

## Moments of Truth

| Moment | Definition | PYB Moment |
|---|---|---|
| **Zero Moment** | First discovery | Seeing the Meta ad or Instagram reel |
| **First Moment** | First direct interaction | Landing on the website |
| **Second Moment** | Product experience | Using the photobooth at the event |
| **Ultimate Moment** | Sharing experience | Posting photos, writing a review, recommending to friends |

**Priority optimization**: First Moment (website) → Zero Moment (ads) → Ultimate Moment (review/referral)

## Psychographic Profiles - PYB Customers

### The Perfectionist Planner (Magic Mirror / Wedding)
- **Values**: Quality, status, perfection
- **Motivation**: "My wedding has to be perfect"
- **Fear**: "What if something goes wrong?"
- **Message**: Emphasize professionalism, equipment quality, on-site guidance
- **Proof needed**: Wedding testimonials, premium event photos

### The Fun Organizer (Magic Mirror / Corporate)
- **Values**: Team experience, wow-factor, looking good to management
- **Motivation**: "I want people to talk about this event"
- **Fear**: "What if it's boring or unprofessional?"
- **Message**: Emphasize wow-factor, previous corporate events, seamless setup
- **Proof needed**: Corporate client logos, event highlight videos

### The Budget-Smart Celebrator (Party Booth)
- **Values**: Fun, value-for-money, simplicity
- **Motivation**: "I want something fun without breaking the bank"
- **Fear**: "Is it worth spending money on?"
- **Message**: Emphasize affordability, ease, fun-factor
- **Proof needed**: Party photos, price clarity, "vanaf €199"

### The All-in-One Seeker (DJ + Photobooth Combo)
- **Values**: Convenience, one-stop-shop, stress reduction
- **Motivation**: "I don't want to coordinate multiple vendors"
- **Fear**: "Will one team be as good as two specialists?"
- **Message**: Emphasize the combo value, Spotify playlist optimization, one contact
- **Proof needed**: Full event testimonials showing both DJ and photobooth
