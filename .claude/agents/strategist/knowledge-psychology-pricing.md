# Marketing Psychology - Pricing & Persuasion

## Pricing Psychology Principles

### 1. Anchoring
Present a higher-priced option first to make the target option seem reasonable.

| Tactic | Implementation | PYB Application |
|---|---|---|
| High anchor first | Show most expensive package first in list | Lead with Magic Mirror VIP (€749+) before Basic (€599) |
| "Was / Now" pricing | Show original price crossed out | Seasonal promotions |
| Context anchor | Compare to more expensive alternatives | "Een DJ bureau rekent €500+, bij ons vanaf €149 erbij" |
| Per-unit framing | Break price into smaller units | "Vanaf €149 per avond entertainment" |

### 2. The Decoy Effect (Asymmetric Dominance)
Introduce a strategically inferior option to push buyers toward the target plan.

**How it works**: When choosing between A and B, adding option C (similar to B but slightly worse) makes B look much better.

| Option | Price | Features | Role |
|---|---|---|---|
| Basic | €599 | 4 hours, booth only | Entry option |
| VIP | €749 | Basic + backdrop, carpet, stanchions, luxe props | **Target** (best value) |
| VIP + Guidance | €899 | VIP + team guidance | Premium anchor |

### 3. Charm Pricing
€599 instead of €600 - works for consumer purchases and impulse decisions.

| Context | Use Charm Pricing? | Why |
|---|---|---|
| Party Booth (€199-250) | Yes | Budget-conscious, price-sensitive segment |
| Magic Mirror Basic | Yes (€599) | Already using it |
| Premium add-ons | Yes (€149, €499) | Charm pricing on all customer-facing prices |

### 4. Price Partitioning
Break the total price into components to make it feel smaller.

**Instead of**: "Complete pakket €899"
**Try**: "Magic Mirror €599 + VIP upgrade €149 + DJ €149"

People evaluate each component separately and perceive the total as less painful.

### 5. Loss Aversion
People feel losses ~2x more strongly than equivalent gains.

**Application**:
- "Mis geen moment - boek nu" (Don't miss a moment - book now)
- "Nog maar 3 weekenden beschikbaar in juni" (Only 3 weekends left in June)
- After proposal: "Je datum is nog beschikbaar, maar we kunnen dit niet lang vasthouden"
- Frame add-ons as "what you'll miss" rather than "what you'll get"

### 6. The Endowment Effect
People value things more once they feel ownership.

**Application**:
- Let customers build their own package (configurator on website)
- Send visual mock-up of their photo strip design in the proposal
- "Jouw event met Pic Your Booth" - make them visualize it as already theirs

### 7. Social Proof Hierarchy

| Type | Impact | PYB Application |
|---|---|---|
| Specific numbers | Highest | "200+ events in de Randstad" |
| Named testimonials | High | "Lisa & Tom, bruiloft juni 2025" with photo |
| Peer endorsement | High | "Gekozen door 50+ bruidsparen dit jaar" |
| Star ratings | Medium | Google reviews on website |
| Generic claims | Low | Avoid "trusted by many" without specifics |

## Cialdini's 6 Principles of Persuasion - Applied to PYB

| Principle | How to Apply | Example |
|---|---|---|
| **Reciprocity** | Give value before asking for booking | Free event planning tips on blog, free photo strip design preview |
| **Social Proof** | Show others choosing PYB | Event photos, testimonials, review count, "200+ events" |
| **Authority** | Demonstrate expertise | Professional equipment details (DSLR, studio lighting), years of experience |
| **Scarcity** | Real availability limits | "Nog 2 weekenden beschikbaar in mei" (real calendar data) |
| **Consistency** | Get small commitments first | Quiz/calculator → form → proposal → booking |
| **Liking** | Be relatable and human | Family story, behind-the-scenes, DJ personalities |

## Fogg Behavior Model (B = MAP)

For a booking to happen, three things must align simultaneously:

| Element | Question | PYB Implementation |
|---|---|---|
| **Motivation** | Do they WANT to book? | Event photos showing wow-factor, testimonials, loss aversion |
| **Ability** | Is it EASY to book? | Simple form (5-6 fields max), clear pricing, fast response |
| **Prompt** | Is there a clear TRIGGER? | CTA button visible, "Vraag je voorstel aan", sticky mobile CTA |

**When conversion is low, diagnose which element is failing:**
- High traffic, low form starts → Motivation problem (page isn't compelling)
- Form starts but low completion → Ability problem (form too complex)
- Good page time but no action → Prompt problem (CTA not visible/clear)

## Cognitive Biases in Event Booking

| Bias | Definition | How to Leverage |
|---|---|---|
| **Status Quo Bias** | People default to doing nothing | Make "no photobooth" feel like a loss, not the default |
| **Bandwagon Effect** | People follow what others do | "De populairste keuze voor bruiloften in de Randstad" |
| **Peak-End Rule** | People judge experiences by the peak moment and the end | Ensure photobooth creates peak moments; end with printed photos as keepsake |
| **Availability Heuristic** | Recent examples feel more likely | Show recent events (dated), seasonal content |
| **Choice Overload** | Too many options = no decision | Max 3 packages per product, clear recommended option |
| **Zero-Risk Bias** | People prefer certainty | "Niet tevreden? Geld terug garantie" or "Vrijblijvend voorstel" |

## Objection Handling Psychology

| Objection | Psychology Behind It | Response Strategy |
|---|---|---|
| "Te duur" (Too expensive) | Price without context = sticker shock | Anchor against alternatives, break into per-hour cost, show ROI in memories |
| "We denken er nog over na" (We'll think about it) | Decision avoidance, loss aversion not triggered | Scarcity ("datum loopt vol"), social proof, reduce risk ("vrijblijvend") |
| "We doen het zelf wel" (We'll DIY) | Overconfidence in own ability | Show quality difference (DSLR vs phone), time savings, stress-free |
| "Hebben we niet nodig" (Don't need it) | Status quo bias | Paint the picture of the event WITH vs WITHOUT, use FOMO |
| "Kennen jullie niet" (Don't know you) | Trust deficit | Reviews, social proof, professional imagery, guarantee |
