SPLIT-ENERGY SYSTEM

PicYourBooth DJs use a split-energy identity. Each DJ owns a signature color. This guide focuses on DJ Gianni (coral), but documents how both identities coexist within the same design system.

DJ Gianni - Coral #f0654a - Afro Beats, Caribbean, NL Urban
Milø - Cyan #34d399 - Tech House, House, Deep
  • Each DJ gets their own accent color applied to: name, EQ bars, genre tags, play buttons, social icon hovers, and card photo overlays
  • Shared elements stay neutral: backgrounds use --void (#050508) and --surface (#111116), text uses --white (#ededf0)
  • Violet (#8b5cf6) is the shared brand accent for non-DJ-specific elements: Spotify section, booking wizard, step indicators
  • Gold (#f59e42) is reserved for the nav CTA button "BOEK JE ERVARING" - always pill-shaped, always gold

DESIGN RULES

Non-negotiable rules for maintaining DJ Gianni's visual identity within the PicYourBooth system.

01
Two fonts only: Bebas Neue (display) and Space Grotesk (body). Never use Oswald, Montserrat, or any other typeface.
02
Coral (#f0654a) is Gianni's signature. Always used for: DJ name, EQ bars, genre tag backgrounds, play buttons, social icon hovers.
03
Never mix Gianni coral with Milø cyan on the same component. Each DJ element gets exactly one accent color.
04
Backgrounds are always --void (#050508) or --surface (#111116). No pure black (#000) or dark grays. The system is tinted.
05
Typography is the design. Whitespace is luxury. The G hexagon is the one bold graphic element - everything else stays minimal.
06
The G hexagon is for Gianni's personal brand (covers, social, merch). PYB website UI uses rounded rectangles from the token scale: 6px (sm) to 999px (pill).
07
"Pure Vibes" is Gianni's tagline. "Elke set een feest, elk feest een herinnering" is the profile quote. Both appear consistently.
08
Gradient text (coral > violet > cyan) is for hero titles only. Never on body text, tags, or buttons. Applied via background-clip: text.
09
Dim color variants (10% opacity) are for tag backgrounds and hover states only. Never use full-opacity coral as a background fill on large areas.
10
Gold (#f59e42) is reserved for the nav CTA "BOEK JE ERVARING". It never appears as a section heading or DJ accent color.
