Architecture

How This Works

Every component in the mockups below is built using Shadcn/ui primitives + PYB theme. No custom UI from scratch.

LAYER 1: SHADCN/UI

Radix primitives — Button, Card, Input, Switch, Dialog, Accordion. Accessible, unstyled.

LAYER 2: PYB THEME

CSS variables override Shadcn. Dark base #08080b, gold #f59e42 accent, Bebas Neue + Space Grotesk.

LAYER 3: PYB COMPOSITIONS

Nav, Hero, ServiceCard, PackageBuilder, ProposalForm — built from Shadcn parts.

Component mapping: Nav = NavigationMenu + Sheet + Button · ServiceCard = Card + Badge + Button · PackageBuilder = Card + Switch + Badge + Button · UpgradeRow = Switch + Dialog · ProposalForm = Input + Label + Select + Textarea + Button + Sonner · FAQ = Accordion

Design Principles

One Brand

PicYourBooth is one premium entertainment brand. Photobooth and DJ pages are rooms in the same venue - same typography, same dark base, same components.

Gold is the Accent

Gold carries the photobooth identity: price pills, CTA buttons, nav book button, VIP badges, toggle states. Violet is reserved for DJ cards on the homepage only.

Same Builder, Every Product

Every product page uses the same package builder structure: locked base package, toggle upgrades, running total, single CTA. Consistent UX across services.
