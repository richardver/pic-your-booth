RELEASE CLIP

PRODUCTION PIPELINE

Interactive TikTok release clip skill for DJ Gianni & Milo

01

OVERVIEW

Takes two raw set recordings, analyzes audio peaks and energy, auto-cuts between angles, and renders a punchy 25-second TikTok clip ready to post.

Format
1080 x 1920
9:16 vertical / TikTok
Framerate
30 FPS
H.264 / CRF 18
Duration
25 SEC
750 frames total
Output Files
4 FILES
mp4 + analysis.json + waveform.html + props.json
02

03

PIPELINE

Six automated steps from raw footage to final output.

1
Analyze Audio
Extracts peaks, BPM, energy curve. Outputs analysis.json + waveform.html
python video/tools/audio-analyzer/analyze.py \ --input angle1.mp4 \ --output video/output/<dj>/tiktok/<series>/
2
Copy Footage to Remotion
Both angles placed in public/ for Remotion access
cp angle1.mp4 video/templates/tiktok/release-clip/public/angle1.mp4 cp angle2.mp4 video/templates/tiktok/release-clip/public/angle2.mp4
3
Copy Cover Art
DJ cover image placed in public/ for end card
cp cover.jpg video/templates/tiktok/release-clip/public/cover.jpg
4
Generate Props
Creates props.json with DJ profile, genre config, scene cuts, hook text
python video/tools/generate-props.py \ --dj gianni \ --genre afro \ --series "Summer Sets" \ --hook "This drop hit different" \ --analysis analysis.json
5
Render via Remotion
Renders 750 frames at 1080x1920, H.264, CRF 18
npx remotion render \ video/templates/tiktok/release-clip/src/index.ts \ ReleaseClip \ --props props.json \ --output release-clip.mp4
6
Output to Final Location
All artifacts moved to the DJ's output directory
mv release-clip.mp4 analysis.json waveform.html props.json \ video/output/<dj>/tiktok/<series>/<clip-name>/
04

ANGLE ASSIGNMENT

Auto-cut logic maps each scene to an angle based on energy phase. The drop always uses Angle 2 for maximum impact.

Scene Frames Time Angle
Hook 0 - 90 0 - 3s Angle 1
Build-up 90 - 180 3 - 6s Angle 1
Pre-drop 180 - 240 6 - 8s Angle 2
Drop 240 - 360 8 - 12s Angle 2
Vibe 360 - 600 12 - 20s Angle 1
End card 600 - 750 20 - 25s Cover art
