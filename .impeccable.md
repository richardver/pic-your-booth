## Design Context

### Users
Event planners, couples planning weddings, party organizers, and corporate event coordinators in the Randstad, Netherlands. They arrive via Meta Ads or Google Ads on mobile (70%+). Their job: find reliable, premium entertainment (photobooth and/or DJ) and submit a proposal request with minimal friction. They're evaluating options, often price-sensitive but quality-conscious.

### Brand Personality
Professional. Creative. Fast-moving. — Confident, energetic, approachable, direct. "Geen gelul." Dutch primary language. Family business energy — genuine, personal, passionate.

### Aesthetic Direction
**Dark cinematic base** — events happen at night. The website feels like walking into a VIP event.

- **Theme:** Dark mode only. Background #08080b, surfaces #111116, elevated #18181e
- **Typography:** Bebas Neue (headings, uppercase, display) + Space Grotesk (body, forms). Major Third scale (1.25).
- **Colors:** Gold #f59e42 (photobooth CTAs, prices), Violet #8b5cf6 (DJ CTAs), Coral #f0654a (DJ Gianni), Cyan #34d399 (Milø). No red (#ff494a), no pink (#e839a0).
- **Gold buttons always use black text.** Violet buttons use white text.
- **References:** Premium event photography, cinematic bokeh, glass morphism on cards
- **Anti-references:** Generic template sites, stock photo aesthetics, corporate SaaS layouts, light/bright themes

### Tech Stack
- Next.js 14+ (App Router) + Tailwind CSS + Vercel
- **Shadcn/ui** as component library — themed via CSS variables with PYB dark tokens
- Radix UI primitives for accessibility
- Bebas Neue + Space Grotesk from Google Fonts

### Component Library
Shadcn/ui components installed in `src/components/ui/`. PYB custom compositions built on top: Nav, Hero, ServiceCard, ProofBar, PackageBuilder, UpgradeRow, ProposalForm, DJProfile, BookingWizard, FAQ, Testimonial. Full spec in `.claude/agents/designer/knowledge-components-full.md`.

### Segment Visual Modes
Three visual modes applied via `data-segment` attributes:
- **Photobooth (default):** Gold #f59e42 accent, premium dark, cinematic
- **DJ pages:** Violet #8b5cf6 accent, high-energy, neon feel
- **DJ Gianni profile:** Coral #f0654a accent, warm, hexagon motif
- **Milø profile:** Cyan #34d399 accent, deep, diamond motif

### Design Principles
1. **Conversion-first** — every element serves the proposal form. One primary CTA per viewport.
2. **Mobile-first** — 70%+ traffic is mobile from Meta Ads. 44px touch targets, sticky CTAs, thumb-zone design.
3. **Message match** — ad headline must echo landing page headline. Visitor confirms "right place" in 3 seconds.
4. **Premium without pretension** — dark, cinematic, gold accents — but approachable and direct, not corporate.
5. **Accessibility is non-negotiable** — WCAG AA contrast (4.5:1), focus indicators, semantic HTML, `prefers-reduced-motion`.

### Pricing Display
Charm pricing always: €199, €599, €149 (not round numbers). Gold badge for prices. "MEEST GEKOZEN" badge on Magic Mirror XL. BTW included, never show ex-BTW.

### Accessibility
- WCAG AA minimum (4.5:1 text, 3:1 UI)
- All Shadcn/Radix components handle ARIA, focus management, keyboard nav
- Respect `prefers-reduced-motion`
- Semantic HTML: nav, main, section, footer
- Touch targets: 44x44px minimum
