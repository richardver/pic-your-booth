Auto Set Builder

Auto Set Builder sequences DJ sets from your library using harmonic mixing, BPM progression, and energy arc presets — the same techniques professionals use, automated in seconds.

It scores every possible track transition across 4 weighted criteria (harmonic compatibility, BPM arc, energy arc, BPM smoothness), selects the optimal sequence using a greedy Camelot-guided algorithm, and exports Rekordbox XML or Serato-compatible files. From Spotify playlist to USB-ready sequenced set in one pipeline. Includes fun DJ commentary per transition — opener lines, quality-rated quips, peak energy callouts, and closer lines.

Algorithm

How Does the Sequencing Algorithm Work?

Greedy Camelot-guided selection with energy arc enforcement. Each slot is filled by scoring every remaining candidate across 4 weighted criteria.

1

Filter Library

Select analyzed tracks matching BPM range, playlists, and file existence. Typically 20-200 candidates from a DJ's library.

2

Calculate Slot Count

Set duration ÷ effective track time (track length minus 30-second transition overlap) = number of tracks needed. A 60-minute set needs approximately 20 tracks at 3.5 minutes each.

3

Generate Slot Targets

For each position, interpolate the preset's energy curve and BPM progression curve. The Peak Time preset targets energy 3 at position 0%, rising to 10 at position 65%, easing to 5 at 100%.

4

Greedy Selection

For each slot, score every remaining candidate across 4 criteria: harmonic compatibility with previous track (weight 0.35), BPM proximity to arc target (0.25), energy proximity to arc target (0.25), and BPM smoothness from previous track (0.15). Pick the highest-scoring candidate.

5

Score Transitions

Rate each adjacency: green = tier 1 harmony + ≤3 BPM jump, yellow = tier 2 + ≤6 BPM, red = tier 3-4 or large jump. A well-sequenced set averages 70%+ green transitions.

6

Export

Generate a Rekordbox XML and/or Serato M3U8 playlist referencing the original file locations — no file copying needed. Files are saved directly in output/ with an autoset_ prefix (e.g. autoset_peak-time_60min.xml or .m3u8). Import the M3U8 into Serato via File > Import Crate. Ready for USB.

Interactive wizard or one-line command. Both produce the same output.

$ pconv set auto --minutes 60 --preset peak-time --bpm-min 120 --bpm-max 130 -t rekordbox
