Features

Everything a DJ Needs

From Spotify to the booth. Every step automated, every format supported.

BPM Detection

librosa beat tracking with auto-normalization to DJ range (60–180 BPM). Halves or doubles to avoid false readings from complex rhythms.

Key Detection

Krumhansl-Schmuckler algorithm over CQT chroma features. Returns key name + confidence score for all 24 major/minor keys.

Camelot Wheel

Automatic mapping to Camelot notation (1A–12B). Compatible key lookup for harmonic mixing — same number, ±1, or relative major/minor.

Mix Point Detection

Energy-based analysis finds where the intro ends and outro begins. Beat-snapped, confidence-scored, written as Rekordbox hot cues A (green) and B (blue).

Rekordbox XML

Full collection + playlist XML with BPM, key, Tonality, and POSITION_MARK hot cues. Import once, sync to USB, ready for CDJs.

Serato Compatible

M3U8 playlist export for Serato crate import (File > Import Crate). ID3v2.3 tags, CBR validation, genre separator checks. Files pass Serato's format scanner without manual fixing.

DJ Set Builder

Build sets from your library. Filter by BPM range, Camelot key (with harmonic neighbors), playlist, or search. Auto-generates set names.

Auto Set Builder

Automated DJ set sequencing using harmonic mixing, BPM progression, and 5 energy arc presets. Exports Rekordbox XML or Serato M3U8 playlist directly — no file copying. Includes fun DJ commentary per transition. Learn more →

DJ Research

Search Spotify by artist or DJ name and download their top tracks — no playlist URL needed. The wizard lets you choose between a Spotify playlist or artist research.

Energy Analysis

Estimates track energy (1–10) from RMS loudness (40%), spectral brightness (30%), and onset density (30%). Powers the Auto Set Builder's energy arc matching.

FAT32 Safe

All filenames stripped of illegal characters, accents transliterated via Unidecode, paths capped at 255 chars. Works on every CDJ and controller.

Pipeline

How It Works

Seven stages from URL to USB. Fully parallel downloads, sequential analysis, optional auto-sequencing, one XML at the end.

1

Fetch Playlist Metadata

Scrapes the Spotify embed page (open.spotify.com/embed/playlist/...) to extract track titles, artists, URIs, durations, and artwork URLs. No API key required — bypasses Spotify's restricted Development Mode.

2

Download Audio

4 parallel workers via spotdl (which wraps yt-dlp + ffmpeg). Each track is matched on YouTube, downloaded as audio, and converted to MP3 at your chosen bitrate. Skips tracks already in the library.

3

Analyze BPM, Key & Mix Points

librosa loads each file at 22050 Hz mono. Beat tracking detects BPM, CQT chroma + Krumhansl-Schmuckler finds the key, and RMS energy analysis locates mix-in/mix-out points. All results are beat-grid snapped.

4

Tag MP3s

mutagen writes ID3v2.3 tags: title, artist, album, year, genre, cover art, BPM (TBPM), key (TKEY), Camelot (TXXX:CamelotKey), and mix points (TXXX:MixInPoint / MixOutPoint).

5

Export Rekordbox XML

Generates a Rekordbox-compatible XML with full collection + playlist. Includes POSITION_MARK hot cues for mix points (green A = mix in, blue B = mix out). Drop the XML into Rekordbox and sync to USB.

6

Organize & Archive

Files are renamed with track numbers, folder gets the BPM range appended (e.g. 2026-02-28_Luca_rekordbox_120-128bpm). Everything is tracked in .library.json for deduplication and set building across sessions.

7

Auto-Sequence Sets

Optional: pconv set auto sequences a DJ set from your library using harmonic mixing (4-tier Camelot distance), BPM progression, and energy arc presets. Greedy algorithm scores every candidate across 4 weighted criteria. Generates Rekordbox XML and/or Serato M3U8 playlist referencing original files — no file copying.

Command Line
# Quick download
$ pconv download URL

# Full analysis + mix points
$ pconv download --analyze-mix-points URL

# Auto set builder
$ pconv set auto --minutes 60 --preset peak-time

# Manual set builder
$ pconv set new
$ pconv set add --bpm 120-128
$ pconv set add --key 8A
$ pconv set save

# Library
$ pconv library list --sort bpm
$ pconv library search "afrobeats"

Under the Hood

Tech Stack

Python CLI, battle-tested audio libraries, no cloud dependencies.

spotdl + yt-dlp

Audio sourced from YouTube. Patched for Spotify API field removals.

librosa + NumPy

BPM, key, and mix point analysis. All local, no API calls.

mutagen

ID3v2.3 tagging. TBPM, TKEY, TXXX, APIC cover art.

Click + Rich

CLI framework + terminal UI with progress bars and tables.

Unidecode

Accent transliteration for FAT32-safe filenames on CDJs.

ffmpeg

Audio transcoding backend. MP3, FLAC, WAV output.
