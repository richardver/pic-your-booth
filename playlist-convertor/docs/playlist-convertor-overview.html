Features

Everything a DJ Needs

From Spotify to the booth. Every step automated, every format supported.

BPM Detection

librosa beat tracking with auto-normalization to the DJ range (60–180 BPM). Halves or doubles to avoid false readings from complex rhythms.

Key Detection

Krumhansl-Schmuckler algorithm over CQT chroma features. Returns key name + confidence score for all 24 major/minor keys.

Camelot Wheel

Auto-mapping to Camelot notation (1A–12B). Compatible key lookup for harmonic mixing — same number, ±1, or relative major/minor.

Mix Point Detection

Energy-based analysis finds where the intro ends and outro begins. Beat-snapped, confidence-scored, written as Rekordbox hot cues.

Rekordbox XML

Full collection + playlist XML with BPM, key, Tonality, and POSITION_MARK hot cues. Import once, sync to USB, ready for CDJs.

Serato Compatible

ID3v2.3 tags, CBR validation, genre separator checks. Files pass Serato's format scanner without manual fixing.

DJ Set Builder

Build sets from your library. Filter by BPM range, Camelot key (with harmonic neighbors), playlist name, or free-text search.

FAT32 Safe

Filenames stripped of illegal chars, accents transliterated via Unidecode, paths capped at 255 chars. Works on every CDJ and controller.

Auto Set Builder

Greedy Camelot-guided sequencing with 5 energy arc presets. Scores transitions across 4 criteria: harmonic (35%), BPM arc (25%), energy arc (25%), smoothness (15%).

Energy Analysis

Estimates track energy 1–10 from RMS loudness (40%), spectral centroid (30%), onset density (30%). Powers energy arc matching in auto sets.

Pipeline

How It Works

Seven stages from URL to USB. Parallel downloads, sequential analysis, optional auto-sequencing, one XML at the end.

1

Fetch Playlist Metadata

Scrapes Spotify's embed page to extract track titles, artists, URIs, durations and artwork. No API key needed — bypasses restricted Development Mode.

2

Download Audio

4 parallel workers via spotdl (wraps yt-dlp + ffmpeg). Each track matched on YouTube, converted to MP3. Skips tracks already in library.

3

Analyze BPM, Key & Mix Points

librosa loads at 22050 Hz mono. Beat tracking for BPM, CQT chroma + Krumhansl-Schmuckler for key, RMS energy for mix-in/mix-out points.

4

Tag MP3s

ID3v2.3 tags: title, artist, album, cover art, BPM, key, Camelot, and mix points as TXXX frames.

5

Export Rekordbox XML

Collection + playlist XML with POSITION_MARK hot cues. Green A = mix in, blue B = mix out. Import and sync to USB.

6

Organize & Archive

Files renamed with track numbers, folder gets BPM range. Tracked in .library.json for dedup and set building.

7

Auto-Sequence Sets

Optional: pconv set auto sequences a set using 4-tier Camelot harmony, BPM progression, and energy arc presets. Outputs sequenced files + XML.

CLI
# Quick download
$ pconv download URL

# Full analysis + mix points
$ pconv download --analyze-mix-points URL

# Auto set builder
$ pconv set auto --minutes 60 --preset peak-time

# Manual set builder
$ pconv set new
$ pconv set add --bpm 120-128
$ pconv set add --key 8A
$ pconv set save

Configuration

Mix Point Tuning

All constants in config.py. Adjust to match your genre and mixing style.

ENERGY_THRESHOLD0.70

Fraction of body median energy to trigger. Lower for ambient, raise for EDM.

SUSTAIN_BEATS8

Consecutive beats above threshold to confirm a crossing point.

SMOOTHING4.0s

RMS smoothing window. Larger ignores short breaks in energy.

MIN_DURATION60s

Tracks under this length are skipped. No mix points on interludes.

FALLBACK16 beats

Default intro/outro when energy is flat or detection fails.

POSITION33/67%

Mix-in clamped to first third, mix-out to last third of track.
