# Video Marketing - Format & Distribution Guide

## Platform Specs

| Platform | Aspect Ratio | Max Length | Ideal Length | Resolution |
|---|---|---|---|---|
| YouTube (long-form) | 16:9 | 12 hours | 8-15 min | 1920x1080+ |
| YouTube Shorts | 9:16 | 3 min | 30-60 sec | 1080x1920 |
| TikTok | 9:16 | 10 min | 15-60 sec | 1080x1920 |
| Instagram Reels | 9:16 | 15 min | 15-30 sec | 1080x1920 |
| Instagram Feed | 1:1 or 4:5 | 60 min | 30-60 sec | 1080x1080 |
| Instagram Stories | 9:16 | 60 sec | 15 sec | 1080x1920 |
| Facebook Video | 16:9 or 1:1 | 240 min | 1-3 min | 1920x1080 |

## Platform Content Strategy

| Platform | Content That Performs | Tone | Hook Window |
|---|---|---|---|
| YouTube | Tutorials, deep dives, vlogs | Educational, personality-driven | 5 seconds |
| YouTube Shorts | Quick tips, reactions, BTS, teasers | Punchy, pattern-interrupt | 1 second |
| TikTok | Trends, demos, storytelling, raw/authentic | Authentic, unpolished, entertaining | 0.5-1 second |
| Instagram Reels | Aesthetic demos, transformations, tips | Polished but relatable | 1 second |
| Facebook | Event videos, albums, reviews | Community-focused | 3 seconds |

## Production Tiers

### Tier 1 - Phone Production (€0-500) - Best for PYB
| Element | Spec |
|---|---|
| Camera | Smartphone (iPhone 13+ or equivalent) |
| Audio | Lavalier mic (€15-30) or wireless (€50-100) |
| Lighting | Natural light or ring light (€20-50) |
| Editing | CapCut (free), InShot |
| Best for | TikTok, Reels, Stories, Shorts |

### Tier 2 - Semi-Professional (€500-5,000)
| Element | Spec |
|---|---|
| Camera | Mirrorless (Canon EOS - PYB already has the 650D!) |
| Audio | Shotgun mic or USB condenser |
| Lighting | 2-3 point setup |
| Editing | DaVinci Resolve (free), Premiere Pro |
| Best for | YouTube, website hero videos, highlight reels |

## Content Repurposing Waterfall for PYB

```
Event Highlight Video (2-3 min, Facebook/YouTube)
│
├─→ 3-5 Short Clips (15-60 sec) → TikTok, Reels, Shorts
│
├─→ Key Frames → Quote graphics for Instagram, Facebook
│
├─→ Behind-the-scenes clips → Stories, TikTok
│
├─→ DJ set clips → DJ personal accounts
│
├─→ Best photos → Instagram carousel, Google Business
│
└─→ Customer testimonial clips → Website, ads, social proof
```

### PYB Clip Selection Criteria

| Criteria | Why |
|---|---|
| Guest reactions ("wow" moments) | Most shareable, emotional impact |
| Photo result reveals | Shows product quality |
| Dance floor energy | Sells the DJ combo |
| Setup/teardown timelapse | Satisfying content, shows professionalism |
| Customer testimonial | Social proof, converts viewers |
| Behind the decks (DJ POV) | Authentic, platform-native for TikTok |

## Thumbnail Design (YouTube / Facebook)

| Element | Best Practice |
|---|---|
| Dimensions | 1280x720px (16:9) |
| Face | Include expressive face (38% more clicks) |
| Text | 3-5 words max, bold, high contrast |
| Branding | Consistent PYB colors/font |
| Emotion | Convey excitement, surprise, energy |
| Test at phone size | Most common viewing context |

## Video SEO (YouTube)

| Element | Implementation |
|---|---|
| Title | Front-load keyword, under 60 chars, compelling hook |
| Description | Keyword-rich summary first 2 lines, then full 200-500 words |
| Tags | 5-10 relevant: "photobooth huren", "bruiloft entertainment" |
| Thumbnail | Custom, face + text, high contrast |
| Captions | Upload SRT file (don't rely on auto-captions) |
| Timestamps | Add chapters in description |
| Hashtags | 3-5 in description |

## Distribution Workflow

| Step | Action | Timeline |
|---|---|---|
| 1 | Upload full video to YouTube/Facebook | Day 0 |
| 2 | Post 3-5 short clips (Reels, TikTok, Shorts) | Days 1-5 (stagger) |
| 3 | Share best clip on DJ personal accounts | Day 1 |
| 4 | Embed on website (event gallery or blog) | Day 0-1 |
| 5 | Include in email follow-up to leads | Next send |
| 6 | Pin best-performing clip to profiles | Ongoing |
| 7 | Use top clips as ad creative | When proven organic |

## Video Performance Metrics

| Metric | Benchmark | Action if Below |
|---|---|---|
| Completion rate (short-form) | >60% | Improve hook, shorten length |
| Completion rate (long-form) | >40% | Better pacing, tighter editing |
| Engagement rate | >3% | More emotional/shareable content |
| Thumbnail CTR (YouTube) | >5% | Redesign thumbnail |
| Share rate | Growing per post | Create more "wow" moments |
