The 4C Creative Framework

1

Catch

Pattern interrupt in first 1-3 seconds. Bold text overlay, unexpected visual, direct question.

2

Connect

Relate to audience pain point. Show you understand their event planning challenge.

3

Convince

Proof, features, social evidence. Testimonials, event footage, before/after results.

4

Close

Clear CTA aligned with funnel stage. "Vraag een offerte aan", "Bekijk pakketten".



Testing Priority Order

  • 1. Hook / Opening -- Biggest impact on performance
  • 2. Value Proposition -- Benefit vs feature vs experience
  • 3. Social Proof -- Numbers vs names vs testimonials
  • 4. CTA -- Direct vs soft, urgency levels
  • 5. Format -- Static vs video vs carousel

Run each variation min 3-5 days. Need 100+ clicks per variation for reliable CTR data.

CPL (Cost Per Lead) €15 - €40 €8 - €20 Target: under €25 avg across campaigns

Scaling Decision Framework

Signal Action
CPA below target for 5+ days Scale budget by 20%
CPA at target, stable Hold, test new creative
CPA 10-30% above target Refresh creative, check frequency
CPA 30%+ above target Reduce budget, diagnose issue

Frequency > 6.0 (retargeting) Reduce budget or expand window

The 0.5 Second Rule

Je hebt een halve seconde. De headline moet SCHREEUWEN zonder te schreeuwen. Groot, helder, onmiddellijk duidelijk. Als iemand moet nadenken, ben je ze kwijt.

VignetteDonkere randen trekken oog naar centruminset box-shadow 60px op .ad-visual
