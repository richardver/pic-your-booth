# Landing Pages — Forms, Personalization & Optimization

## Form Optimization

### The Golden Rule
Every additional form field reduces conversion by approximately 4-7%.

### PYB Proposal Request Form — Recommended

| Field | Type | Required | Why |
|---|---|---|---|
| Naam | Text | Yes | Personal follow-up |
| E-mail | Email | Yes | Send proposal |
| Telefoon | Tel | Yes | Quick follow-up |
| Datum event | Date picker | Yes | Check availability |
| Type event | Dropdown | Yes | Tailor proposal |
| Bericht | Textarea | No | Extra context (optional) |

**6 fields maximum. No more.**

### Form UX Principles

| Principle | Implementation |
|---|---|
| **Progressive disclosure** | Show only essential fields first; ask more in the proposal email |
| **Inline validation** | Real-time feedback as fields are completed |
| **Smart defaults** | Pre-select most common event type |
| **Appropriate input types** | `tel` for phone (shows numpad), `email` for email, `date` for date |
| **Error messages** | Specific: "Vul een geldig e-mailadres in" not "Fout" |
| **Submit button copy** | "Ontvang je voorstel" (not "Verstuur" or "Submit") |
| **Sub-CTA text** | "Binnen 24 uur reactie • Vrijblijvend" |
| **Thank-you page** | Confirmation + expectation: "We nemen binnen 24 uur contact op!" |

### Multi-Step Form Alternative
For complex bookings, consider 2 steps:
1. **Step 1**: Event date + type (low commitment)
2. **Step 2**: Contact details (invested now, more likely to complete)

Progress bar showing "Stap 1 van 2" reduces anxiety.

## Message Match — Critical for PYB Funnel

The #1 conversion killer is mismatch between ad and landing page.

### PYB Message Match Rules

| Ad Content | Landing Page Must Show |
|---|---|
| "Magic Mirror XL Photobooth" + event video | Hero with same visual style + "Magic Mirror XL" in headline |
| "Photobooth huren vanaf €200" | Price visible above fold: "Vanaf €200" |
| "DJ + Photobooth combo" | Combo package front and center, not buried |
| Testimonial ad from wedding couple | Same testimonial on landing page + more wedding proof |
| Before/after event footage | Same energy in hero image/video |

**Test**: Can a visitor who clicked the ad confirm they're in the right place within 3 seconds?

## Checkout / Booking Optimization

### Reduce Booking Friction

| Friction Point | Solution |
|---|---|
| Unclear pricing | Show "Vanaf €X" on every package — don't hide prices |
| No availability info | Date picker that shows available dates (or "Check beschikbaarheid") |
| Too many choices | Max 3 packages per product line, one highlighted as "Meest gekozen" |
| No trust signals | Google reviews widget, testimonial near form, "200+ events" counter |
| Slow response | Promise and deliver "<24 uur reactie" — automate confirmation email |
| No risk reversal | "Vrijblijvend voorstel" (no-obligation proposal) prominently displayed |

### The Proposal Email — Conversion Psychology

The proposal email is a critical conversion step. Apply psychology:

| Element | Psychology | Implementation |
|---|---|---|
| Personalized greeting | Liking principle | "Hoi [naam], leuk dat je een event plant!" |
| Visual mock-up | Endowment effect | Show their event type with PYB setup |
| Package recommendation | Choice architecture | Highlight one "Aanbevolen" package |
| Testimonial | Social proof | Include a testimonial matching their event type |
| Clear pricing | Reduce anxiety | No hidden costs, transparent total |
| Scarcity | Loss aversion | "Je datum is nog beschikbaar" |
| Easy next step | Ability (Fogg) | "Klik hier om te bevestigen" — one action |
| Time limit | Urgency | "We houden je datum 7 dagen vast" |

## Email Sequences for PYB

### Post-Form-Submit Nurture (if not immediately booked)

| Email | Timing | Subject | Content |
|---|---|---|---|
| 1 | Immediate | "Je voorstel van Pic Your Booth" | Proposal with packages, pricing, CTA to confirm |
| 2 | Day 2 | "Heb je nog vragen, [naam]?" | Address common objections, FAQ link |
| 3 | Day 5 | "Zo was het bij [similar event]" | Testimonial + event photos matching their type |
| 4 | Day 8 | "Je datum is populair" | Scarcity trigger, availability reminder |
| 5 | Day 14 | "Laatste check: nog interesse?" | Soft close, offer to call/chat |

### Post-Event Sequence

| Email | Timing | Subject | Content |
|---|---|---|---|
| 1 | Day 1 | "Bedankt voor de geweldige avond!" | Thank you, ask how it went |
| 2 | Day 3 | "Je foto's staan klaar!" | Gallery link, shareable |
| 3 | Day 5 | "Zou je ons willen reviewen?" | Google review link, easy 1-click |
| 4 | Day 7 | "Ken je iemand die ook een feest plant?" | Referral offer (€50 voor jullie allebei) |
| 5 | Month 3 | "Alweer bijna [seizoen] — plan je iets?" | Seasonal rebooking prompt |
