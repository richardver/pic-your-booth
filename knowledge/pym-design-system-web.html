Architecture
How This Design System Works

shadcn/ui — Component Layer

shadcn/ui provides accessible, composable components built on Radix UI primitives and Tailwind CSS.

Stack:

  • Radix UI for accessibility & behavior
  • Tailwind CSS for styling
  • CSS variables for theming
  • Lucide for icons (20px / 1.5px stroke)

Theming shadcn/ui → PYM

Override shadcn/ui's default CSS variables with PYM brand tokens in globals.css. Components inherit the theme automatically.

PYM overrides:

  • Font: system → Poppins (single font, no exceptions)
  • Colors: neutral → PYM 9-color palette
  • Radius: 6px → 12px default
  • Dark mode: zinc → Neutral #121212 (no color cast on photos)
  • Icons: default → Lucide 20px / 1.5px

Cross-Platform Consistency

Web and Flutter share the same brand tokens. Visual consistency across all PYM surfaces.

Shared with Flutter:

  • 9 brand colors (same hex values)
  • Spacing scale (4px base)
  • Border radius (8 / 12 / 16 / 999)
  • Typography (Poppins 400 / 500 / 600)
  • Icon system (Lucide)
  • Light & dark theme semantics

Tailwind CSS Config
Drop this into your globals.css. All shadcn/ui components will inherit the PYM theme automatically.
/* globals.css — PYM theme for shadcn/ui */

@layer base {
  :root {
    --background: 240 14% 97%;     /* #f7f7fa */
    --foreground: 244 55% 13%;     /* #100e34 */
    --card: 30 18% 94%;            /* #f4f1ef */
    --card-foreground: 244 55% 13%;
    --popover: 0 0% 100%;
    --popover-foreground: 244 55% 13%;
    --primary: 0 100% 64%;         /* #ff494a */
    --primary-foreground: 0 0% 100%;
    --secondary: 24 18% 88%;       /* #e8e0db */
    --secondary-foreground: 244 55% 13%;
    --muted: 30 18% 94%;
    --muted-foreground: 244 10% 29%;
    --accent: 30 18% 94%;
    --accent-foreground: 244 55% 13%;
    --destructive: 0 100% 64%;
    --destructive-foreground: 0 0% 100%;
    --border: 24 18% 88%;
    --input: 24 18% 88%;
    --ring: 244 55% 13%;
    --radius: 0.75rem;             /* 12px */
  }

  .dark {
    --background: 0 0% 7%;         /* #121212 */
    --foreground: 240 14% 97%;
    --card: 0 0% 11%;              /* #1c1c1c */
    --card-foreground: 240 14% 97%;
    --popover: 0 0% 11%;
    --popover-foreground: 240 14% 97%;
    --primary: 0 100% 64%;
    --primary-foreground: 0 0% 100%;
    --secondary: 0 0% 16%;         /* #2a2a2a */
    --secondary-foreground: 240 14% 97%;
    --muted: 0 0% 11%;
    --muted-foreground: 240 10% 78%;
    --accent: 0 0% 11%;
    --accent-foreground: 240 14% 97%;
    --destructive: 0 100% 64%;
    --destructive-foreground: 0 0% 100%;
    --border: 0 0% 16%;
    --input: 0 0% 16%;
    --ring: 240 14% 97%;
  }
}

@layer base {
  body { font-family: 'Poppins', sans-serif; }
}
